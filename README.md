# Full-Stack E-Commerce

This is a Next.js full-stack e-commerce starter (sanity + Stripe). It was pushed to this repository by the project owner.

Quick start

1. Install dependencies:

   - npm
     ```bash
     npm install
     ```

   - or using bun
     ```bash
     bun install
     ```

2. Copy environment variables from `.env.example` to `.env` and fill values.

3. Run the development server:

```bash
npm run dev
```

Building

```bash
npm run build
```

Notes
- This repo contains Sanity studio and Next.js frontend code.
- See `package.json` for available scripts.
