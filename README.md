Quick start

1. Install dependencies:

   - npm
     ```bash
     npm install
     ```

   - or using bun
     ```bash
     bun install
     ```

2. Copy environment variables from `.env.example` to `.env` and fill values.

3. Run the development server:

```bash
npm run dev
```

Building

```bash
npm run build
```
